Upper Triangular Matrix
(By Maneesh Shakhwar)

What is a Upper Triangular Matrix?

A Upper triangular matrix is a type of square matrix
where all elements below the main diagonal are zero.
This can be achieved through Gaussian elimination.

Example of Converting a Matrix to Upper Triangular Form

Consider the matrix A:

               | 2   3   1 |
           A = | 4   7   3 |
               | 6   8   9 |

Upper Triangular matrix

Make the (2,1) element zero using Row Elimination

                R2 = R2 - (2) × R1
    
                              | 2   3   1 |
            After elimination:| 0   1   1 |
                              | 6   8   9 |

Make the (3,1) element zero using Row Elimination

                R3 = R3 - (3) × R1
        
                              | 2   3   1 |
            After elimination:| 0   1   1 |
                              | 0   -1   6|

Make the (3,2) element zero using Row Elimination

                R3 = R3 - (-1) R2
        
                              | 2   3   1 |
            After elimination:| 0   1   1 |
                              | 0   0   7 |

Final Upper Triangular Matrix (U)

                | 2   3   1 |
            U = | 0   1   1 |
                | 0   0   7 |
