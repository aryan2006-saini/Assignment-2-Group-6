Lower Triangular Matrix
(By Aryan Saini)

What is a Lower Triangular Matrix?

A Lower triangular matrix is a type of square matrix
where all elements above the main diagonal are zero.
This can be achieved through Gaussian elimination.

Example of Converting a Matrix to Lower Triangular Form

Consider the matrix A:

               | 2   3   1 |
           A = | 4   7   3 |
               | 6   8   9 |

Lower Triangular Matrix (L)

Step 1.1: Make the (2,3) element zero using Row Elimination

                R2 = R2 - (0.33) × R1
    
                              | 2   3     1 |
            After elimination:| 2  4.33   0 |
                              | 6   8     9 |

Step 1.2: Make the (1,3) element zero using Row Elimination

                R1 = R1 - (0.111) × R3
        
                              |1.33 2.11  0 |
            After elimination:|2    4.33  0 |
                              |6    8     9 |

Step 1.3: Make the (1,2) element zero using Row Elimination

                R1 = R2 - (0.487) R2
        
                              |0.36  0    0|
            After elimination:|2    4.33  0|
                              |6    8     9|

Final Lower Triangular Matrix (U)

                  |0.36  0    0|
            L =   |2    4.33  0|
                  |6    8     9|
