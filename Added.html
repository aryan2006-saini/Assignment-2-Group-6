    What is L, D, U Decomposition?

    The LU decomposition of a matrix is a factorization of a
    matrix into the product of a lower triangular matrix (L),
    a diagonal matrix (D), and an upper triangular matrix (U).
    This can be done using Gaussian elimination.

    Example of Finding L, D, and U from Matrix A

    Consider the matrix A:

                  | 2   3   1 |
          A = | 4   7   3 |
              | 6   8   9 |

    We will now perform Gaussian elimination to obtain L, D, and U step-by-step:

    Step 1:Lower Triangular Matrix (L)

    Step 1.1: Make the (2,3) element zero using Row Elimination

                    R2 = R2 - (0.33) × R1
    
                              | 2   3     1 |
            After elimination:| 2  4.33   0 |
                              | 6   8     9 |

    Step 1.2: Make the (1,3) element zero using Row Elimination

                    R1 = R1 - (0.111) × R3
        
                              |1.33 2.11  0 |
            After elimination:|2    4.33  0 |
                              |6    8     9 |

    Step 1.3: Make the (1,2) element zero using Row Elimination

                    R1 = R2 - (0.487) R2
        
                              |0.36  0    0|
            After elimination:|2    4.33  0|
                              |6    8     9|

    Final Lower Triangular Matrix (U)

                      |0.36  0    0|
            L =   |2    4.33  0|
                  |6    8     9|

    Step 2:Diagonal Matrix (L)

    Step 2.1: Make the (2,1) element zero using Row Elimination

                    R2 = R2 - (2) × R1
    
                              | 2   3   1 |
            After elimination:| 0   1   1 |
                              | 6   8   9 |

    Step 2.2: Make the (3,1) element zero using Row Elimination

                    R3 = R3 - (3) × R1
        
                              | 2   3   1 |
            After elimination:| 0   1   1 |
                              | 0   -1   6|

    Step 2.3: Make the (1,2) element zero using Row Elimination

                    R1 = R2 - (3) R2
        
                              |2   0  -2|
            After elimination:|2   1   1|
                              |0  -1   6|

    Step 2.4: Make the (3,2) element zero using Row Elimination

                    R3 = R3 - (-1) R2
        
                              |2   0  -2|
            After elimination:|0   1   1|
                              |0   0   7|

    Step 2.5: Make the (1,3) element zero using Row Elimination

                    R1 = R1 - (-0.286) R3
        
                              |2   0   0|
            After elimination:|0   1   1|
                              |0   0   7|

    Step 2.6: Make the (2,3) element zero using Row Elimination

                    R2 = R2 - (0.143) R3
        
                              |2   0   0|
            After elimination:|0   1   0|
                              |0   0   7|

    Final Diagonal Matrix (D)

                | 2   0   0 |
        D = | 0   1   0 |
            | 0   0   7 |

    Step 3: Upper Triangular matrix

    Step 3.1: Make the (2,1) element zero using Row Elimination

                    R2 = R2 - (2) × R1
    
                              | 2   3   1 |
            After elimination:| 0   1   1 |
                              | 6   8   9 |

    Step 3.2: Make the (3,1) element zero using Row Elimination

                    R3 = R3 - (3) × R1
        
                              | 2   3   1 |
            After elimination:| 0   1   1 |
                              | 0   -1   6|

    Step 3.3: Make the (3,2) element zero using Row Elimination

                    R3 = R3 - (-1) R2
        
                              | 2   3   1 |
            After elimination:| 0   1   1 |
                              | 0   0   7 |

    Final Upper Triangular Matrix (U)

                    | 2   3   1 |
            U = | 0   1   1 |
                | 0   0   7 |

    Final Decomposition Results

    The final L, D, and U matrices are:

    Lower Triangular Matrix (L):

                    |0.36  0    0|
            L = |2    4.33  0|
                |6    8     9|

    Diagonal Matrix (D):

                    | 2   0   0 |
            D = | 0   1   0 |
                | 0   0   7 |

    Upper Triangular Matrix (U):

                    | 2   3   1 |
            U = | 0   1   1 |
                | 0   0   7 |
